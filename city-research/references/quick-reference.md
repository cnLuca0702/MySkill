# 快速参考指南

## 技能名称
**comprehensive-city-research** - 综合城市调研技能

## 核心能力
1. **17个搜索引擎**：8个国内 + 9个国际
2. **9章固定报告框架**：结构化、标准化
3. **数据交叉验证机制**：多源对比、可信度分层
4. **严格文字风格规范**：无列表、无表格、无emoji

---

## 快速启动流程

### 第1步：确定调研对象
- 城市名称
- 区/县名称（如适用）
- 调研深度（全面/专项）

### 第2步：执行基础搜索
```bash
# 行政区划
web_search({"query": "{城市名称} 行政区划", "count": 10})

# 地理环境
web_search({"query": "{城市名称} 地形地貌 海拔 气候", "count": 10})

# 战略定位
web_search({"query": "{城市名称} 在城市群 战略定位", "count": 10})
```

### 第3步：执行数据搜索
```bash
# 人口数据
web_search({"query": "{城市名称} 第七次人口普查", "count": 10})

# 经济数据
web_search({"query": "{城市名称} 2024年GDP 统计公报", "count": 10})

# 房价数据
web_fetch({"url": "https://m.creprice.cn/district/{区代码}.html?city={城市代码}"})

# 教育
web_search({"query": "{城市名称} 教育 学位 2024", "count": 10})
```

### 第4步：验证数据
- 优先使用政府官网数据
- 至少2-3个来源交叉验证
- 标注数据时间和来源

### 第5步：撰写报告
- 遵循9章框架
- 严格遵守文字风格规范
- 段落长度150-300字

## 报告框架速查

| 章节 | 核心内容 | 必需数据 |
|------|---------|---------|
| 1. 区域概况与战略定位 | 行政区划、地理、战略 | 成立、面积、人口、定位 |
| 2. 交通圈与区域协同 | 交通网络、覆盖、协同 | 高铁、高速、人口、产业 |
| 3. 人口结构 | 人口规模、年龄、民族 | 人口、结构、增长 |
| 4. 经济发展 | GDP、产业、投资 | GDP、产业、投资、消费 |
| 5. 城市更新 | 老旧小区、城中村、基建 | 项目、投资、薄弱环节 |
| 6. 房地产 | 房价、片区、居住 | 价格、片区、地标 |
| 7. 公共服务 | 教育、医疗、养老 | 学位、床位、机构 |
| 8. 城市定位 | 文化、商业、旅游、产业 | 四层面定位及路径 |
| 9. 发展展望 | 目标、路径、举措 | 量化目标、实施路径 |

## 文字风格速查

### ✅ 推荐用词
**动词**: 深耕、主导、构建、整合、打通、聚焦、深化、推进、实施、完善
**形容词**: 扎实、完整、精准、协同、持续、深入、全面、显著、突出
**名词**: 底层逻辑、核心价值、全链条、业务闭环、落地路径、承载区、集聚区

### ❌ 严禁事项
- 列表符号（章节标题除外）
- 表格呈现
- emoji表情符号
- 空洞表述/堆砌术语/夸张浮夸
- 汉字数字（全部用阿拉伯数字）

### 📝 段落规范
- 长度：150-300字
- 结构：主题句 → 支撑内容 → 承转衔接
- 逻辑：递进、连贯、流畅

## 应急搜索技巧

### 找不到最新数据
1. 搜索 "2024年" 或 "2025年"
2. 使用 filetype:pdf 限定
3. 访问政府统计局官网

### 找不到具体数字
1. 使用 site:gov.cn 限定
2. 搜索 "统计公报" 或 "统计年鉴"
3. 使用 WolframAlpha 计算

### 找不到政府官网
1. 搜索 "{城市} 人民政府"
2. 搜索 "{城市拼音}.gov.cn"
3. 从上级政府网站找到链接

### 找不到房价数据
1. 使用 creprice.cn（中国房价行情）
2. 使用 anjuke.com（安居客）
3. 使用 gotohui.com（聚汇数据）

## 常见问题

### Q1: 报告应该多长？
**A**: 根据数据丰富程度，15000-50000字不等。深度调研建议30000字以上。

### Q2: 数据冲突怎么办？
**A**: 
1. 标注数据来源和时间
2. 分析冲突原因
3. 说明数据差异
4. 优先采用最新数据

### Q3: 如何确保数据时效性？
**A**: 
1. 优先搜索2024-2025年数据
2. 搜索关键词包含年份
3. 查找最新发布的统计公报
4. 标注数据采集时间

### Q4: 如何提高搜索效率？
**A**: 
1. 使用 filetype:pdf 限定文件类型
2. 使用 site:gov.cn 限定政府网站
3. 使用多个关键词组合搜索
4. 使用17个搜索引擎全面覆盖

### Q5: 报告如何交付？
**A**: 
1. 保存为 Markdown 格式
2. 附加数据来源清单
3. 标注报告生成时间
4. 提供可编辑版本
