# 综合城市调研技能 (comprehensive-city-research)

## 📋 技能概述

本技能整合了 **multi-search-engine**（17个搜索引擎）和 **city-research**（城市调研报告框架）两大核心能力，用于系统化深度调研中国城市及城区的发展情况，输出专业、完整、结构化的城市调研报告。

### 核心优势
- ✅ **17个搜索引擎覆盖**：8个国内 + 9个国际
- ✅ **无需API密钥**：直接URL访问，开箱即用
- ✅ **9章固定报告框架**：结构化、标准化
- ✅ **严格文字风格规范**：无列表、无表格、无emoji
- ✅ **数据交叉验证机制**：多源对比、可信度分层

## 🚀 快速开始

### 第1步：确定调研对象
```
城市名称：如"银川金凤区"
调研深度：全面/专项
```

### 第2步：执行数据收集
```bash
# 基础信息
web_search({"query": "{城市名称} 行政区划", "count": 10})

# 人口经济
web_search({"query": "{城市名称} 2024年统计公报", "count": 10})

# 房价数据
web_fetch({"url": "https://m.creprice.cn/district/{区代码}.html?city={城市代码}"})
```

### 第3步：验证数据
- 优先使用政府官方数据
- 多源数据交叉验证
- 注明数据时间节点

### 第4步：撰写报告
- 遵循9章固定框架
- 严格遵守文字风格规范
- 段落长度控制在150-300字

## ✍️ 文字风格规范

### 严禁事项
- ❌ 条目式/列表符号/分点编号（章节标题除外）
- ❌ 表格呈现/使用"|"符号
- ❌ emoji 表情符号
- ❌ 空洞表述/堆砌术语/夸张浮夸

### 数字规范
- 所有数字使用阿拉伯数字，不使用汉字数字
- 百分比使用%符号
- 金额单位统一为元、万元、亿元
- 面积单位统一为平方米、亩、平方公里

### 推荐用词
- 动词：深耕、主导、构建、整合、打通、聚焦、深化
- 形容词：扎实、完整、精准、协同、持续、深入
- 名词：底层逻辑、核心价值、全链条、业务闭环

### 段落规范
- 长度：150-300字
- 结构：主题句 → 支撑内容 → 承转衔接
- 风格：逻辑递进、连贯流畅

## 🔍 高级搜索技巧

### 站内搜索
```
site:gov.cn {城市名称} 统计公报
site:{城市拼音}.gov.cn {关键词}
```

### 文件类型搜索
```
{城市名称} filetype:pdf
{城市名称} filetype:doc
{城市名称} filetype:xlsx
```

### 精确匹配
```
"{关键词}"
-{排除词}
```

## 📋 数据验证原则

### 可信度层级
1. **第一层**：政府统计公报
2. **第二层**：政府官方网站
3. **第三层**：权威媒体（人民网、新华网）
4. **第四层**：专业平台（房产平台、统计平台）

### 验证流程
```
数据收集 → 多源对比 → 识别差异 → 分析原因 → 确定可信度 → 标注来源
```
