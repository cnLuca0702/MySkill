# 综合城市调研技能

## 技能概述

本技能整合了 multi-search-engine（17个搜索引擎）和 city-research（城市调研报告框架）两大核心能力，用于系统化深度调研中国城市及城区的发展情况，输出专业、完整、结构化的城市调研报告。适用于政府决策参考、企业投资评估、学术研究等场景。

**核心优势：**
- 17个搜索引擎覆盖：8个国内 + 9个国际
- 无需API密钥，直接URL访问
- 9章固定报告框架
- 严格文字风格规范
- 数据交叉验证机制

---

## 触发条件

当用户提出以下类型请求时触发本技能：
- 帮我深度调研某个城市/城区
- 某个城市/城区的全面发展分析
- 城市投资环境深度评估
- 城市产业/人口/经济综合分析
- 城市规划与发展战略研究

## 核心方法

### 方法2：数据交叉验证机制

为确保数据准确性，采用多源数据交叉验证：

#### 验证原则
1. 优先采用政府官方数据
2. 多源数据交叉验证
3. 注明数据时间节点
4. 数据差异需说明原因

#### 验证流程
```
数据收集 → 多源对比 → 识别差异 → 分析原因 → 确定可信度 → 标注来源
```

#### 可信度层级
- **第一层**：政府统计公报（统计局、发改委）
- **第二层**：政府官方网站（人民政府、各部门）
- **第三层**：权威媒体（新华网、人民网、地方日报）
- **第四层**：专业平台（房产平台、统计平台）

## 工作流程

### 第一阶段：基础信息收集

#### 1.1 行政区划与地理区位

**搜索关键词组合：**
- `{城市名称} 区划 历史 建区时间`
- `{城市名称} 地形地貌 海拔 气候 降水`
- `{城市名称} 政府驻地 路网结构`

**必查数据项：**
- 成立时间、地理位置、相邻区域、总面积
- 行政区划（镇/街道数量、行政村/社区数量）
- 地形地貌、海拔范围、气候类型
- 年平均气温、年降水量

**搜索引擎优先级：**
1. 百度（政府信息）
2. 必应中国（官方资讯）
3. 百度百科（基础信息）
4. 政府网站（权威数据）

#### 1.2 城市战略定位

**搜索关键词组合：**
- `{城市名称} 在城市群 战略定位`
- `{城市名称} 承担功能 连接区域`
- `{城市名称} 一带一路 节点城市`

**必查数据项：**
- 在所属城市群中的定位和使命
- 连接哪些区域、承担什么功能
- 核心城区在城市整体战略中的功能

**搜索关键词示例：**
- `银川 黄河流域生态保护 中心城市`
- `银川 丝绸之路经济带 节点`
- `{城市名称} 区域协同 枢纽`

---

### 第二阶段：人口与经济数据

#### 2.1 人口结构

**搜索关键词组合：**
- `{城市名称} 第七次人口普查 数据`
- `{城市名称} 常住人口 2024 年龄结构`
- `{城市名称} 出生率 死亡率 自然增长率`

**必查数据项：**
- 常住人口数量及同比增长
- 出生率、死亡率、自然增长率
- 年龄结构（0-14岁、15-59岁、60岁以上）
- 民族构成（汉族、回族、其他少数民族）

**搜索引擎优先级：**
1. 政府统计局
2. 百度（人口普查数据）
3. 聚汇数据（历年数据）
4. 红黑人口库（人口统计）

#### 2.2 经济数据

**搜索关键词组合：**
- `{城市名称} 2024 年 GDP 统计公报`
- `{城市名称} 地区生产总值 产业结构`
- `{城市名称} 固定资产投资 社会消费品零售`

**必查数据项：**
- 地区生产总值及增速
- 三次产业增加值及占比
- 人均地区生产总值
- 固定资产投资总额及增速
- 社会消费品零售总额及增速

**搜索关键词示例：**
- `{城市名称} 统计公报 2024 filetype:pdf`
- `{城市名称} 地区生产总值 site:gov.cn`
- `{城市名称} 经济发展报告`

### 第七阶段：报告撰写

#### 7.1 遵循9章固定框架

严格按照以下9章结构撰写报告，章节顺序不可调整：

1. 区域概况与战略定位
2. 两小时交通圈与区域协同分析
3. 人口结构与社情特征
4. 经济发展与产业布局
5. 城市更新与空间优化
6. 房地产市场与居住生态
7. 公共服务与民生保障
8. 城市定位与发展建议（文化/商业/旅游/产业四层面）
9. 发展展望与实施路径

#### 7.2 遵守文字风格规范

**严禁事项：**
- 禁止使用条目式/列表符号/分点编号（章节标题除外）
- 禁止使用表格呈现/使用"|"符号
- 禁止使用 emoji 表情符号
- 禁止空洞表述/堆砌术语/夸张浮夸

**数字规范：**
- 所有数字使用阿拉伯数字，不使用汉字数字
- 百分比使用%符号
- 金额单位统一为元、万元、亿元
- 面积单位统一为平方米、亩、平方公里

**推荐用词：**
- 动词：深耕、主导、构建、整合、打通、聚焦、深化、推进、实施、完善
- 形容词：扎实、完整、精准、协同、持续、深入、全面、显著、突出
- 名词：底层逻辑、核心价值、全链条、业务闭环、落地路径、承载区、集聚区

**语言特点：**
- 专业性：术语准确、逻辑严密、避免口语化
- 稳重性：客观陈述、数据说话、不夸张
- 说服力：结构清晰、论证充分、结论有力
- 实践性：具体案例、量化成果、可落地

**段落结构：**
- 主题句 → 支撑内容（案例/数据） → 承转衔接
- 段落间逻辑递进，连贯流畅
- 每段长度不少于 150 字

## 使用示例

### 示例1：调研银川金凤区

#### 步骤1：基础信息收集

```bash
# 使用百度搜索行政区划
web_fetch({"url": "https://www.baidu.com/s?wd=银川金凤区 行政区划"})

# 使用必应中国搜索地理信息
web_fetch({"url": "https://cn.bing.com/search?q=银川金凤区 地形地貌 海拔"})

# 使用政府网站查询官方数据
web_fetch({"url": "http://www.ycjinfeng.gov.cn/"})
```

#### 步骤2：人口经济数据

```bash
# 搜索人口普查数据
web_fetch({"url": "https://www.baidu.com/s?wd=银川金凤区 第七次人口普查"})

# 搜索GDP统计公报
web_fetch({"url": "https://www.baidu.com/s?wd=银川金凤区 统计公报 2024 filetype:pdf"})

# 使用聚汇数据验证
web_fetch({"url": "https://gdp.gotohui.com/show-1135438"})
```

#### 步骤3：房产数据

```bash
# 使用中国房价行情
web_fetch({"url": "https://m.creprice.cn/district/JF.html?city=yc"})

# 使用安居客房价数据
web_fetch({"url": "https://m.anjuke.com/yin/trendency/8076/"})
```

#### 步骤4：撰写报告

按照9章框架，结合收集的数据，撰写符合文字风格规范的调研报告。

## 注意事项

1. **数据时效性**：优先使用最新年份的数据，注明数据时间节点
2. **数据准确性**：多源交叉验证，数据差异需说明原因
3. **报告篇幅**：根据数据丰富程度，报告不少于15000字
4. **定位建议**：需结合两小时交通圈分析，具有可操作性
5. **城市更新**：需包含主要工作和薄弱环节两个方面
6. **发展展望**：需有量化目标（人口、GDP、产业占比等）
