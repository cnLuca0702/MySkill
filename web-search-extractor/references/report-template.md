## 内容提取指南

### 相关政策提取

**政策类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 法律法规 | 全国人大制定的法律 | 《中华人民共和国旅游法》 |
| 行政法规 | 国务院制定的法规 | 《旅行社条例》 |
| 部门规章 | 各部委发布的规章 | 《研学旅行服务规范》 |
| 行业标准 | 行业标准、规范、指南 | LB/T 054-2025《研学旅游服务要求》 |
| 地方政策 | 地方政府发布的相关政策 | 各省市研学旅行实施意见 |
| 指导意见 | 政府发布的指导性文件 | 《关于推进中小学生研学旅行的意见》 |

**提取要素**：
- 政策名称（完整官方名称）
- 发布时间（精确到年月日）
- 发布部门/机构（全称）
- 核心内容/要点（2-3条）
- 政策影响/意义

**政策时序整理**：
- 按发布时间从早到晚排列
- 标注政策层级（国家级/省部级/行业标准）
- 突出重要政策节点
- 体现政策演进脉络

### 典型案例提取

**案例类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 品牌出海案例 | 国际化推广案例 | 汉文化交流研学之旅 |
| 行业影响力案例 | 获得行业认可 | 中国旅游产业影响力案例 |
| 优秀实践案例 | 成功实施的项目 | 省级优秀案例 |
| 教育创新案例 | 教育模式创新 | 数字化教学创新 |
| 创新案例 | 技术/模式创新 | 跨界融合案例 |
| 负面案例 | 需要改进的问题 | "游而不学"现象 |

**提取要素**：
- 案例名称
- 申报/实施单位
- 荣誉/认可（入选XX案例、获得XX奖项）
- 核心特色/亮点（2-3点）

### 相关企业提取

**企业类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 龙头企业 | 行业领先、营收规模大 | 中凯国际、世纪明德 |
| 中坚企业 | 一定规模、区域性品牌 | 宝贝走天下 |
| 新兴企业 | 新创立、快速成长 | 本地化服务商 |
| 研究机构 | 科研院所、咨询机构 | 艾媒咨询 |

**提取要素**：
- 企业名称
- 市场地位/特点（净利润居首、营收规模领先）
- 主要业务/特色（亲子研学、营地教育）
- 营收规模（如有具体数据）

### 新闻网站
- 新华网、人民日报等：使用 lynx 提取正文
- 注意识别日期、作者信息

### 学术网站
- 优先提取摘要和结论
- 识别论文标题、作者、发表时间

### 行业报告
- 提取关键数据和图表描述
- 识别报告来源、发布时间

### 博客/自媒体
- 验证信息可靠性
- 标注发布时间
